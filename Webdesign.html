Historija Web Designa

Historija web dizajna započinje s nastankom World Wide Weba (WWW) 1989. godine, kada je britanski informatičar Tim Berners-Lee stvorio prvi web server i web preglednik.
U početku, web stranice bile su jednostavne, uglavnom sastavljene od teksta i ponekad s nekoliko statičnih slika.
Kroz 1990-e godine, web dizajn se razvijao paralelno s tehnološkim napretkom.
HTML (Hypertext Markup Language) postao je standardni jezik za izradu web stranica, omogućavajući strukturiranje sadržaja pomoću oznaka i hiperveza.
U ovom ranom periodu, većina web stranica bila je statična, bez dinamičkog sadržaja ili interaktivnosti.
S pojavom CSS-a (Cascading Style Sheets) sredinom 1990-ih, web dizajn dobio je novu dimenziju.
CSS omogućio je odvajanje dizajna i sadržaja, što je olakšalo kontrolu nad izgledom web stranica.
To je dovelo do razvoja složenijih i estetski privlačnijih dizajna.
